# CSOT GenAI Submissions

Submissions for the CSOT GenAI/Agentic Track 2026.

## Structure

```
week_1/         Multi-turn terminal chatbot with streaming and history compaction
week_2/         Research agent with web search, page fetch, AlphaXiv MCP, and Textual TUI
```

## Setup

Each week has its own `requirements.txt` and `.env.example`. Copy `.env.example` to `.env`, fill in your API keys, then install dependencies:

```bash
pip install -r requirements.txt
```

See `week_2/project/SUBMISSION.md` for full setup and run instructions.
